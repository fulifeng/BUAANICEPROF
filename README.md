# 学生心中的北航好导师
这里是北航学子自发分享，推荐，整理的北航好导师列表。我们希望这里的信息能够帮到有兴趣到北航交流，深造的学子如愿找到靠谱的导师，做出有价值的项目，研究。

# 如何推荐
## Pull request
直接编辑对应学院目录（比如：计算机学院）下对应导师的markdown文件（比如：怀进鹏.md）。如果学院目录或者导师文件缺失，请手动创建。
## Email
将推荐信息发送到Contributor邮箱（比如：hennande@gmail.com）
